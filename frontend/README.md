# HireMind AI — Next.js

Pixel-identical port of the HireMind AI landing page to Next.js (App Router, JavaScript) using **Tailwind CSS v4**, **Framer Motion**, and **Lucide React**.

## Install

```bash
npm install
```

## Run dev server

```bash
npm run dev
```

Open http://localhost:3000

## Build for production

```bash
npm run build
npm start
```

## Stack

- Next.js 15 (App Router, JavaScript only — no TypeScript)
- Tailwind CSS v4 (CSS-first config in `src/app/globals.css`, identical tokens to the source design)
- Framer Motion for animations
- Lucide React for icons

Styling matches the source 1:1 — the same CSS variables, glass utilities, gradients, and component markup are used.
